## 1. 빠른 시작 (5분 완성)

### 1.1 처음 켰을 때 화면 구성

앱을 열면 **대시보드(시작 화면)**가 나타납니다. 왼쪽 사이드바에 환자 목록, 오른쪽 본 영역에 선택된 환자의 진행 상태가 표시됩니다.

- **웹·Electron Standalone**: 별도 로그인 없이 바로 사용합니다.
- **Electron Intranet** `[인트라넷 전용]`: 로그인 화면이 먼저 나타납니다. 계정과 기기 승인이 필요합니다 (→ [10장](#10-인트라넷-모드-it-담당자관리자)).

### 1.2 첫 환자 처음부터 끝까지 — 최소 필수 흐름

```
[대시보드]
    │
    ▼  "새 환자" 버튼 클릭
[1단계: 기본정보]
    │  환자 인적사항 + 직업력 1개 이상 입력
    │
    ▼
[2단계: 상병 입력]
    │  ICD 코드 또는 상병명 입력 (1개 이상)
    │  → 모듈 자동 추천 표시
    │
    ▼
[3단계: 모듈 선택]
    │  추천 모듈 확인 후 필요 시 추가/제거
    │
    ▼
[부위별 신체부담 평가 탭]
    │  선택된 모듈마다 탭 생성 (예: 무릎 · 어깨)
    │  각 탭: 직업별 노출 수치 입력 → 우측 결과 패널 확인
    │
    ▼
[종합소견]
    │  상병별 추가 지표(KLG 등급 등) 입력
    │  업무관련성 판정 (높음 / 낮음) 선택
    │
    ▼
[내보내기]
    │  EMR 형식 엑셀 또는 PDF로 저장
    │
    ▼
[완료]
```

### 1.3 이전 데이터 이어서 쓰기

- **웹·Standalone**: 왼쪽 사이드바에서 환자 목록을 클릭하면 이전 작업 내용이 복원됩니다. 앱이 닫혀도 자동저장이 유지됩니다.
- **인트라넷** `[인트라넷 전용]`: 서버에 저장된 데이터는 다른 PC에서도 불러올 수 있습니다. (자동저장은 서버 동기화 방식 — 로컬 자동저장은 비활성화됨)

## 2. 화면 구성 개요

### 2.1 대시보드 — 전체 레이아웃

```
┌──────────────────────────────────────────────────────────────┐
│  [헤더]  앱 제목          [AI 분석] [설정] [계정 `인트라넷`]   │
├──────────────┬───────────────────────────────────────────────┤
│              │                                               │
│  [사이드바]  │  [본 영역]                                    │
│              │                                               │
│  • 새 환자   │  환자 선택 전:                                 │
│  • 엑셀 불러  │    → 대시보드 (환자 통계 카드,                  │
│    오기      │       최근 작업 목록)                           │
│              │                                               │
│  [환자 목록] │  환자 선택 후:                                 │
│  (필터·정렬) │    → 단계 표시줄 + 현재 단계 입력 화면           │
│              │                                               │
│              │    [기본정보] [상병 입력] [모듈 선택]            │
│              │    [무릎] [어깨] ... [종합소견] [AI 분석]        │
│              │                                               │
└──────────────┴───────────────────────────────────────────────┘
```

### 2.2 사이드바 (환자 목록·필터·정렬)

- 상단 검색창: 이름·환자번호 실시간 검색
- 필터: 완료/미완료, 모듈 보유 여부
- 정렬: 최근 수정 순, 이름 순
- `[인트라넷 전용]` 범위 토글: **내 환자** ↔ **전체** (관리자·의사만 전체 조회 가능)

### 2.3 단계(스텝) 표시줄

환자 선택 후 화면 상단에 표시됩니다.

```
기본정보 → 상병 입력 → 모듈 선택 → [무릎] → [어깨] → 종합소견 → AI 분석
```

- 완료된 단계는 체크 표시
- 미완료 항목이 있는 단계는 주황색 표시
- 단계를 클릭하면 해당 단계로 바로 이동

### 2.4 인트라넷 모드 추가 요소 `[인트라넷 전용]`

- 헤더 우상단: 로그인 계정 정보, 로그아웃 버튼
- 관리자 계정에는 **관리자 콘솔** 버튼 추가 표시

## 3. 신규 환자 등록 — 3단계 등록 마법사

### ⚠️ 핵심 개념: 직업력(공통) vs. 부위별 신체부담(모듈) 분리

이 앱의 가장 중요한 구조적 특징입니다.

**직업력(공통)** — 기본정보 탭에서 입력
- 직종명, 사업장, 근무 기간, 주당 근무일수, 연간 근무일수
- 한 번 입력하면 **모든 모듈이 공유**합니다.

**부위별 신체부담** — 각 모듈 탭에서 별도 입력
- 무릎 탭: 쪼그려앉기 분/일, 중량물 kg/일 등
- 어깨 탭: 오버헤드 시간/일, 반복동작 시간/일 등
- 같은 직업이라도 **부위마다 수치가 다릅니다.** 모듈 탭을 열어 각각 입력해야 합니다.

> 예시: "철근 조립공, 2010~2020년" → 기본정보 탭에 한 번 입력  
> 무릎 탭: 쪼그려앉기 60분/일 → 어깨 탭: 오버헤드 2시간/일 → 각각 따로 입력

---

### 3.1 1단계: 기본정보

#### 인적사항 필수 항목

| 항목 | 설명 | 필수 |
|------|------|------|
| 환자번호 | 병원 EMR 환자번호 | 선택 |
| 성명 | 한글 이름 | 필수 |
| 성별 | 남/여 | 필수 |
| 생년월일 | YYYY-MM-DD | 필수 |
| 신장 (cm) | 척추 MDDM 계산에 사용 | 권장 |
| 체중 (kg) | 척추 MDDM 계산에 사용 | 권장 |
| 부상·발병일 | | 선택 |
| 평가일 | 기본값: 오늘 | 권장 |

#### 직업력 입력

하단 **직업력** 섹션에서 직업을 추가합니다. 한 환자에 여러 직업 추가 가능합니다.

| 항목 | 설명 |
|------|------|
| 직종명 | 자유 텍스트 |
| 사업장 | 자유 텍스트 |
| 시작/종료 연도 | 현재 재직 중이면 종료 연도 비워둠 |
| 주당 근무일수 | 기본 5일 |
| 연간 근무일수 | 기본 250일 |

**직업 프리셋 사용법**: 직업력 입력창 옆 돋보기 아이콘(프리셋 검색)을 클릭하면 이전에 저장된 직업 부담 수치를 불러올 수 있습니다. 직종명으로 검색 후 적용하면 해당 모듈의 신체부담 항목이 자동 채워집니다.

#### 진료기록 및 특이사항

- **진료기록(소견)**: 의뢰서 내용, 환자 증상 기재
- **타과 회신**: 정형외과/신경과/재활의학과 각각 별도 입력란
- **특이사항**: 고혈압, 당뇨, 이전 방문 이력 등
- **특이사항 텍스트**: 기타 자유 기재

#### 평가기관 기본값

병원명, 진료과, 의사명은 설정에서 기본값을 저장해두면 새 환자마다 자동 입력됩니다 (→ [9장 설정](#9-설정)).

---

### 3.2 2단계: 상병 입력

#### ICD 코드와 상병명 입력

"상병 추가" 버튼을 클릭하면 상병 입력 행이 생성됩니다. 각 행에:

- **ICD 코드**: M17.3 처럼 입력 (자유 텍스트)
- **상병명**: ICD 코드에 맞는 상병명 입력
- **좌/우/양측 구분**: 해당하는 항목 선택
- **모듈 지정** (v5.1.1 신규): 상병별로 어느 모듈로 평가할지 드롭다운으로 수동 지정 가능. 비워두면 시스템이 자동 추천.

#### 모듈 자동 추천 로직

입력한 ICD 코드·상병명을 기반으로 시스템이 적절한 모듈을 추천합니다.

| ICD 범주 | 추천 모듈 |
|----------|----------|
| M17.x (무릎 골관절염), M22.x, M23.x | 무릎 (슬관절) |
| M75.x (어깨 병변), M77.0 | 어깨 |
| M77.0/1, M70.2/3 (팔꿈치) | 팔꿈치 |
| M65.x, M67.x (수부·손목) | 손목/손가락 |
| M51.x, M47.x (요추) | 척추 (요추 MDDM) |
| M54.2, M50.x (경추) | 경추 |

→ 자동 추천이 부정확하면 모듈 지정 드롭다운에서 직접 선택하세요.

#### 수동 모듈 지정 (v5.1.1)

상병 행의 모듈 드롭다운에서:
- **자동 추천** (기본): 시스템이 ICD 코드 기반으로 결정
- **없음**: 이 상병을 어느 모듈과도 연결하지 않음
- **무릎 / 어깨 / 팔꿈치 / 손목 / 척추 / 경추**: 해당 모듈로 명시적 지정

---

### 3.3 3단계: 모듈 선택

2단계에서 추천된 모듈 목록이 표시됩니다.

- **자동 추천 모듈**: 체크박스로 활성/비활성화
- **추가 모듈**: 목록 하단에서 다른 모듈을 수동 추가
- **조합 예시**: 무릎+척추 (퇴행성 무릎 + 요추부 추간판 탈출 복합), 팔꿈치+손목 (상과염 + 수근관증후군 복합)

선택된 모듈마다 이후 단계에서 탭이 생성됩니다.

## 4. 부위별 신체부담 평가

### 4.1 공통 구조 — 좌측 입력 / 우측 결과 패널

모든 모듈 탭은 동일한 2-패널 레이아웃을 사용합니다.

```
┌─────────────────────┬───────────────────────────┐
│   좌측: 입력 패널    │   우측: 결과 패널          │
│                     │                           │
│  직업별 카드         │  계산 결과                 │
│  ┌───────────────┐  │  • 기여도·등급             │
│  │ 직업명 [탭]   │  │  • 임계값 비교표           │
│  │ 노출 수치 입력 │  │  • 종합 판정 문구          │
│  └───────────────┘  │                           │
└─────────────────────┴───────────────────────────┘
```

직업이 여러 개인 경우 좌측 패널에 직업별 탭(또는 카드)이 생성됩니다. 각 탭에서 해당 직업의 부담 수치를 별도로 입력합니다.

---

### 4.2 무릎 (슬관절)

**기반 기준:** 국내 업무관련성 판단 기준

#### 입력 항목

| 항목 | 단위 | 비고 |
|------|------|------|
| 쪼그려앉기 | 분/일 | 무릎 90° 이상 굴곡 자세 |
| 중량물 취급 | kg/일 평균 | 운반·들기 포함 |
| 계단 오르내리기 | 예/아니오 | |
| 무릎 비틀림 동작 | 예/아니오 | 회전·비틀기 반복 |
| 출발·정지 반복 | 예/아니오 | 잦은 가속/감속 동작 |
| 좁은 공간 작업 | 예/아니오 | 무릎 굽힘 강요 환경 |
| 무릎 접촉·충격 | 예/아니오 | 무릎이 바닥·구조물에 닿음 |
| 뛰어내리기 | 예/아니오 | 높이에서 뛰어내림 |

#### 결과 해석

- **기여도**: 직업별로 계산된 신체부담의 각 요인별 기여 점수
- **부담 등급**: 고도 / 중등도상 / 중등도하 / 경도 (4단계)
- **누적 부담 판정**: 전체 직력을 합산한 종합 판정 문구

---

### 4.3 어깨 (견관절 BK2117)

**기반 기준:** 독일 직업병 BK2117 — 누적 노출 시간 기반 임계값 비교

#### 입력 항목 (단위: 시간/일)

| 항목 | 단위 | 설명 |
|------|------|------|
| 오버헤드/어깨 높이 이상 작업 | 시간/일 | 팔을 어깨 위로 드는 작업 |
| 반복동작 중간속도 (4~14회/분) | 시간/일 | 중속 반복 작업 시간 |
| 반복동작 고도 (≥15회/분) | 시간/일 | 고속 반복 작업 시간 |
| 중량물(≥20 kg) 취급 | 횟수/일 + 초/회 | 취급 횟수와 1회 소요 시간 분리 입력 |
| 손-팔 진동 (≥3 m/s²) | 시간/일 | 진동 공구 노출 시간 |

> **반복동작 OR 조건**: 중간속도 OR 고도 중 하나만 임계값을 초과해도 반복동작 기준 충족으로 판정합니다.

#### BK2117 누적 노출 임계값

| 노출 유형 | 임계값 |
|-----------|--------|
| 오버헤드 작업 | 3,600시간 |
| 반복동작 중간속도 | 38,000시간 |
| 반복동작 고도 | 9,400시간 |
| 중량물(≥20 kg) 취급 | 200시간 |
| 손-팔 진동 | 5,300시간 |

#### 결과 해석

누적 시간 = 일일 노출 시간 × 연간 근무일수 × 근무 년수로 직력 전체를 합산합니다.

- **기준 초과 ≥1개 항목**: "기준 초과, 누적 신체부담 충분"
- **초과 없으나 복합 노출**: 50%↑ 달성 항목 3개 이상 또는 75%↑ 달성 2개 이상 → "복합 노출 고려, 충분"
- **그 외**: "기준 미달, 불충분"

### 4.4 팔꿈치 (주관절) — Gate-and-Flag 방식

**기반 기준:** 독일 산재보험 BK2101 / BK2103 / BK2105 / BK2106

> Gate-and-Flag 방식: 임계값 점수가 아닌 **위험 요인 조합(flag)**을 신호로 표시하고, 서술형 종합평가 문장을 자동 생성합니다.

#### BK 유형 4가지 및 자동 매핑

| BK 유형 | 질환명 | 핵심 요인 |
|---------|--------|----------|
| BK2101 | 상과병변 / 부착부 건병증 | 반복 동작 주기, 강제 배측굴곡, 회내·회외 반복 |
| BK2103 | 팔꿈치 골관절염 / 박리성 골연골염 | 진동 공구 종류, 1일 진동 사용시간 |
| BK2105 | 팔꿈치 점액낭염 | 팔꿈치 지지·기대기, 직접 압박·마찰·충격 |
| BK2106 | 주관증후군 / 척골신경병변 | 같은 자세 유지, 직접 압박 수준 |

상병 입력 시 ICD 코드에 따라 BK 유형이 자동 배정됩니다. 배정이 부정확하면 드롭다운으로 수동 선택합니다.

#### 직업 × 상병 2차원 입력 구조

좌측 패널에 **직업별 카드**가 생성되고, 각 카드 안에 해당 직업과 연결된 **상병별 엔트리 카드**가 나열됩니다. 상병이 3개이면 직업당 엔트리 카드도 3개입니다.

#### ⚠️ 핵심: 직접 해부학적 연결성

각 상병 엔트리 카드의 맨 첫 번째 항목입니다.

- **"예" 선택 시**: 노출 세부 입력 항목이 모두 펼쳐집니다.
- **"아니오" 선택 시**: 세부 항목이 숨겨집니다. 이 선택 자체가 "직접 연결성 없음"을 의미하므로 추가 입력이 불필요합니다.

> 가장 많은 오류 원인: "아니오"를 선택한 뒤 세부 항목이 나타나지 않는 것을 보고 오류라고 오해하는 경우. 정상 동작입니다.

#### ⚠️ 공통 시간적 선후관계 — 모듈 전체 1회만 입력

우측 결과 패널 최상단에 "시간적 선후관계" 섹션이 있습니다.

- **직업 단위가 아닌 모듈 단위**로 1회만 입력합니다.
- 직업이 2개 이상이어도 두 번째 직업 카드에는 이 항목이 없습니다. 빠뜨린 것이 아닙니다.

| 항목 | 내용 |
|------|------|
| 최근 업무 변화 | 예/아니오 |
| 업무 변화 시기 | 날짜 |
| 증상 발생 간격 | 적절 / 불명확 / 부적절 |
| 휴식 시 증상 호전 | 예/아니오 |

#### BK 유형별 추가 입력 항목

- **BK2101**: 1회 동작 주기(초), 시간당 반복횟수, 단조 반복 패턴, 강제 배측굴곡 여부, 회내·회외 반복 여부
- **BK2103**: 진동 공구 종류(직선/회전/충격), 1일 진동 사용시간(시간), 파지·가압 작업 여부
- **BK2105**: 팔꿈치 기대기 여부, 직접 압박 원인 기재
- **BK2106**: 자세 고정 시간 비율, 직접 압박 수준(없음/경도/중등/심함), 압박 원인 기재

#### 결과 해석

우측 패널에 직업별·상병별 Summary Card가 표시됩니다.

- **Flag pill**: 감지된 위험 요인 항목 (예: "고빈도 반복", "배측굴곡 자세", "지속 압박")
- **Narrative**: 평가 근거 서술 자동 생성 문단
- **종합평가 문장**: 업무관련성 근거 요약 1~2문장

### 4.5 손목/손가락 (수관절) — Gate-and-Flag 방식

**기반 기준:** BK2113 / BK2101 / BK2103 / BK2106

팔꿈치와 동일한 Gate-and-Flag 구조입니다. 아래 차이점에 주목하세요.

- **직접 해부학적 연결성**: 팔꿈치와 동일하게 작동 (→ [4.4절](#44-팔꿈치-주관절--gate-and-flag-방식))
- **공통 시간적 선후관계**: 손목 모듈 전체 1회 입력 (팔꿈치와는 독립적)
- **BK2113 추가 항목**: 손목 굴곡·신전 반복, 파지 작업 특성, 수근관압박 관련 항목

### 4.6 경추 (BK2109)

**기반 기준:** 독일 직업병 BK2109 — 경추 누적 부담 평가

#### 노출 유형 2가지

경추 평가는 두 가지 노출 유형 중 해당하는 유형을 선택하여 입력합니다.

| 유형 | 기준 | 주요 입력 |
|------|------|----------|
| 어깨 하중 운반 | 하중 ≥ 40 kg | 운반 하중(kg), 1일 운반 시간(시간) |
| 비중립 목 부하 | 1일 ≥ 1.5~2시간 | 비중립 자세 유형, 1일 노출 시간 |

#### 작업 단위 입력

척추(요추)와 유사하게 **작업 단위별**로 입력합니다:
- 작업명, 노출 유형, 노출 수치를 작업별로 추가
- 직업이 여러 개이면 직업별 탭으로 분리됩니다

### 4.7 척추 (요추 MDDM)

**기반 기준:** 독일 Mainz-Dortmund Dose Model (MDDM) — 요추 압박력 누적 선량 평가

#### 작업 추가·편집

좌측 패널에서 **직업별 탭** 안에 작업(Task) 목록을 관리합니다.

- "작업 추가" 버튼 → 작업명 입력 → 자세 코드 선택 → 중량물·빈도·시간 입력
- 작업 순서는 드래그앤드롭으로 변경 가능 (v5.1.0 추가)

#### 자세 코드 (G1~G11)

각 작업에 대해 대표 자세 코드 하나를 선택합니다. G1이 가장 가벼운 자세, G11이 가장 심한 자세입니다. 상세 설명은 [부록 B](#부록-b-척추-자세-코드-g1g11-설명표) 참조.

#### 입력 항목 (작업당)

| 항목 | 단위 | 비고 |
|------|------|------|
| 작업명 | 텍스트 | |
| 자세 코드 | G1~G11 | 부록 B 참조 |
| 중량물 | kg | 취급 물체 무게 |
| 빈도 | 회/시간 또는 회/일 | |
| 시간 | 분/일 또는 비율 | |
| 보정계수 (F1~F4) | 선택 | 비대칭·비틀림 등 추가 부하 보정 |

#### ⚠️ 직업이 2개 이상일 때

탭으로 분리된 각 직업별로 작업을 따로 입력합니다. 직업 A의 작업은 직업 A 탭에, 직업 B의 작업은 직업 B 탭에 입력해야 합니다. 탭을 이동하며 입력하세요.

#### ⚠️ 4,000N 규칙 (중요)

단일 작업의 추정 압박력이 **4,000 N 이상**이면, 일일 선량이 기준값(2.0 kN·h)에 미달하더라도 해당 작업은 **평생 누적 선량에 포함**됩니다. 중량 하중 작업이 포함된 경우 이 조건을 반드시 확인하세요.

#### 보정계수 (F1~F4) 적용 시점

기본 자세 코드가 포착하지 못하는 추가 부하가 있을 때 적용합니다.

| 보정계수 | 적용 조건 |
|---------|----------|
| F1 | 비대칭 자세 (비틀림 없는 측방 굴곡) |
| F2 | 비틀림 동반 (체간 회전 포함) |
| F3 | 비대칭 + 비틀림 |
| F4 | 불안정한 동적 하중 |

#### 결과 해석 — 3가지 기준 비교

| 판정 기준 | 남성 | 여성 |
|----------|------|------|
| DWS2 | 7.0 MN·h | 3.0 MN·h |
| 독일 법원 기준 | 12.5 MN·h | 8.5 MN·h |
| MDDM | 25 MN·h | 17 MN·h |

결과 패널에 3기준 대비 현재 누적 선량이 막대 그래프로 표시됩니다. 어느 기준을 충족하는지 확인 후 의사 판단으로 업무관련성을 결정합니다.

## 5. 종합소견

모든 모듈 탭 입력 완료 후 **종합소견** 단계로 이동합니다.

### 5.1 상병별 추가 지표 입력

> ⚠️ KLG 등급, Ellman Class 등 진단 지표는 **이 탭에서만** 입력합니다. 신체부담 평가 탭에는 없습니다.

상병 목록이 나열되고 각 상병에 해당하는 추가 지표를 입력합니다.

| 모듈 | 추가 입력 항목 | 좌/우 구분 |
|------|--------------|----------|
| 무릎 | KLG 등급 (0~4) | 좌 / 우 |
| 어깨 | Ellman Class | 좌 / 우 |
| 팔꿈치 | BK 유형 최종 확인 | 좌 / 우 |
| 손목 | BK 유형 최종 확인 | 좌 / 우 |
| 척추 | 수직분포원리, 동반 척추증 여부 | 없음 (첫 진단만 입력) |
| 경추 | — | 없음 |

> **척추 수직분포원리**: 가장 먼저 입력된 척추 관련 상병 행에만 표시됩니다. 상병이 여러 개여도 한 번만 입력합니다.

### 5.2 업무관련성 판정

각 상병별로:

- **높음**: 업무와 질환 사이 인과관계 인정
- **낮음**: 낮다고 판단하는 사유를 자유 텍스트로 기재

### 5.3 복귀 고려사항

직장 복귀 관련 권고 사항을 자유 텍스트로 입력합니다. 복귀 조건, 작업 제한 사항, 추적 관찰 권고 등을 기재합니다.

### 5.4 우측 통합 미리보기 활용

종합소견 탭 우측 패널에 **통합 리포트 초안**이 실시간으로 표시됩니다.

- 모든 모듈의 계산 결과 + 업무관련성 판정 + 복귀 고려사항이 통합된 텍스트
- EMR 붙여넣기용 소견서 초안으로 활용 가능
- AI 분석 전에 이 미리보기로 입력 누락 여부를 확인하는 용도로도 유용합니다

## 7. 내보내기

### 7.1 형식 비교

| 형식 | 용도 | 대상 |
|------|------|------|
| EMR 형식 (.xlsx) | EMR 직접 업로드 또는 기록 보관 | 현재 환자, 선택 환자, 전체 환자 |
| 일괄입력용 (.xlsx) | 나중에 앱으로 재업로드 | 데이터 백업·이전 용도 |
| PDF | 종이 출력·전자 첨부 | 무릎 모듈 활성화 시 가능 |

### 7.2 EMR 형식 내보내기

내보내기 버튼 클릭 후:

- **현재 환자**: 현재 열려 있는 환자 1명의 데이터
- **선택 환자**: 사이드바에서 체크박스로 선택한 환자 여럿
- **전체 환자**: 전체 환자 목록 (.zip 묶음)

출력 파일에는 인적사항, 직업력, 모듈별 평가 결과, 업무관련성 판정, 복귀 고려사항이 포함됩니다.

### 7.3 일괄입력용 (Roundtrip 재업로드)

일괄입력용 엑셀은 앱의 입력 데이터를 75개 열 형식으로 저장합니다. 이 파일을 수정한 후 **엑셀 불러오기**로 다시 업로드하면 앱에 반영됩니다.

활용 사례: 행정직이 기본정보(인적사항·직업력)를 사전 입력한 파일을 전문의에게 전달 → 전문의가 불러와서 신체부담 평가만 추가 입력.

### 7.4 PDF 저장

무릎(슬관절) 모듈이 활성화된 환자는 PDF 저장 옵션이 나타납니다. 브라우저 인쇄 기능을 통해 PDF로 저장합니다.

- **사이드바**: 저장된 환자 목록을 클릭하면 해당 환자 데이터가 열립니다.
- **이전 버전 데이터**: 이전 단일 모듈 버전(예: 무릎만 있던 앱)에서 내보낸 엑셀을 일괄입력용 형식으로 가져올 수 있습니다.

### 8.3 엑셀 일괄입력 `[행정직]`

행정직 담당자가 환자 기본정보를 사전 일괄 입력하는 워크플로입니다.

1. 사이드바 하단 **엑셀 불러오기** 버튼 클릭
2. 일괄입력용 형식의 엑셀 파일 선택
3. 미리보기 확인 후 가져오기 실행

**열 카테고리 (주요 필수·권장 항목)**

| 카테고리 | 주요 열 | 필수 여부 |
|----------|--------|---------|
| 인적사항 | 성명, 성별, 생년월일 | 권장 |
| 직업력 | 직종명_1, 시작년도_1, 종료년도_1, 사업장_1 | 권장 |
| 상병 | ICD코드_1, 상병명_1, 좌우_1 | 권장 |
| 모듈별 수치 | 각 모듈 전용 열 | 선택 |

열 이름이 일치하지 않으면 해당 열은 무시됩니다. 파일을 한 번 내보낸 후 템플릿으로 활용하는 것을 권장합니다.

### 8.4 프리셋 관리

자주 쓰는 직업 노출 수치를 프리셋으로 저장·재사용할 수 있습니다.

- **저장**: 직업 입력 완료 후 프리셋 저장 버튼 → 프리셋명 입력
- **검색**: 직업력 옆 돋보기(프리셋 검색) → 직종명으로 검색
- **적용**: 검색된 프리셋 선택 → 해당 모듈의 신체부담 수치 자동 입력
- **내보내기/가져오기**: 프리셋 관리 모달에서 JSON 파일로 내보내기/가져오기 (다른 PC 이전 등)

2. "승인 대기" 상태의 기기 확인
3. 해당 기기의 **승인** 버튼 클릭
4. 사용자에게 앱 재시작 안내

## 11. 자주 묻는 질문 (FAQ)

**Q. 상병을 잘못 입력했을 때 수정하는 방법은?**

상병 입력(2단계) 탭으로 이동해 해당 상병 행의 수정 아이콘을 클릭합니다. ICD 코드와 상병명을 수정할 수 있습니다. 모듈 지정이 바뀌면 해당 모듈의 입력 데이터가 초기화될 수 있으므로 신중하게 수정하세요.

---

**Q. 직업력을 수정하면 모듈 데이터에 영향이 있는가?**

직업력(기본정보 탭)에서 직업 기간·근무일수를 변경하면 **누적 선량·누적 시간 계산값이 즉시 재계산**됩니다. 척추(MDDM)·어깨(BK2117) 등 누적 시간 기반 모듈에서 결과값이 달라집니다. 직업을 완전히 삭제하면 해당 직업에 귀속된 신체부담 입력 데이터도 함께 삭제됩니다.

---

**Q. 동일 환자를 여러 의사가 동시에 편집하면 어떻게 되는가?** `[인트라넷 전용]`

마지막으로 저장한 데이터가 적용됩니다(Last-Write-Wins). 충돌 감지 시 충돌 해결 다이얼로그가 표시되어 어느 버전을 유지할지 선택할 수 있습니다. 동시 편집은 가급적 피하고, 편집 전 최신 데이터를 불러왔는지 확인하세요.

---

**Q. 이전 버전(단일 모듈) 데이터 불러오기가 가능한가?**

가능합니다. 이전 무릎 전용 앱 또는 척추 전용 앱에서 내보낸 엑셀 파일을 일괄입력용 형식으로 가져오면 대부분의 공통 필드(인적사항, 직업력)가 자동 매핑됩니다. 이전 버전에만 있던 전용 열은 무시됩니다.

---

**Q. 엑셀 일괄입력 열 이름이 불일치할 때 대처 방법은?**

열 이름이 정확히 일치해야 인식됩니다. 불일치 열은 무시됩니다. 내보내기 기능으로 현재 앱의 템플릿 파일을 먼저 생성한 후 해당 파일 형식에 맞춰 데이터를 작성하는 것이 가장 안전합니다.

---

**Q. AI 분석 응답이 없거나 오류가 발생할 때 대처 방법은?**

→ [6.4절 분석 실패 시 대처](#64-분석-실패-시-대처) 참조.

---

**Q. 팔꿈치 모듈을 열었는데 세부 입력 항목이 보이지 않는다.**

"직접 해부학적 연결성" 항목을 **"예"**로 선택해야 세부 노출 항목이 펼쳐집니다. "아니오"를 선택하면 연결성 없음으로 평가되어 세부 항목이 숨겨지는 것이 정상입니다 (→ [4.4절](#44-팔꿈치-주관절--gate-and-flag-방식)).

---

**Q. 척추 결과 패널에서 DWS2/독일 법원/MDDM 3가지 기준 중 어느 것을 기준으로 판정해야 하는가?**

세 기준은 각각 다른 맥락에서 사용됩니다. 국내 산재 심사에서 주로 참고하는 기준은 기관·심사 유형에 따라 다릅니다. 결과 패널에는 세 기준 모두 표시되므로, 의사가 판단 근거로 활용할 기준을 직접 선택하여 종합소견에 기재하세요.
